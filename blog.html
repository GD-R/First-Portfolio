How to Define a Variable:

You need to first delcare a variable and intialize the value. To declare a variable you can use two keywords let or const
For Example: let num = 5
here let is a keyword, num is a variable name and 5 is the value assigned to variable num. When ever if you wish to change the value of variable you can simply write num = 10 so the new value of variable num will be 10. When const keyword is used insted of let then the value assigned during the intialization can not be changed through out tht program.
For Example: const num = 5;
When const is used and in future if you try to change the value of num it will throw an error.
Value of the variable can be anything like integer, word, char or sentences. Below are the ways to difine differnt datatypes.
To define word the Value must be quoted within the double or single quote.
